# Restaurant Clusters & Predictions

An interactive map that clusters Berkeley restaurants using **k-means** and predicts personalized ratings for simulated user personas.

Built with **Python**, **Streamlit**, and **Folium** for the UC Berkeley C88C project.

## Features
- Interactive map visualization with adjustable clusters (k)
- Personalized predicted ratings per user persona
- Category filtering (e.g., Sandwiches, Pizza, etc.)
- Live deployment with Streamlit Cloud
